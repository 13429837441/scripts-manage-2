## 🎯 工具特点
### 1️⃣ 一站式脚本管理
支持多种脚本类型：

- PowerShell (.ps1)
- Python (.py)
- Node.js (.js/.mjs/.cjs)
- 批处理 (.bat/.cmd)
- Shell (.sh)
- 以及 Ruby、PHP、Perl、VBS 等多种脚本
无需在不同终端之间切换，一个界面搞定所有脚本！

### 2️⃣ 一键执行与监控
- ✅ 点击即可执行，无需手动输入命令
- ✅ 实时显示执行状态（运行中/已完成）
- ✅ 完整的执行日志记录
- ✅ 支持手动停止运行中的脚本
### 3️⃣ 灵活的定时任务
支持多种定时方式：

- 间隔执行 ：每N分钟/小时/天执行一次
- 定时执行 ：每天/每周/每月指定时间执行
- Cron表达式 ：支持标准Cron语法，灵活定制
再也不用担心错过定时任务！

### 4️⃣ 可视化界面
采用赛博朋克风格设计，界面酷炫且易用：

- 侧边栏导航，功能一目了然
- 卡片式脚本列表，信息清晰
- 响应式布局，适配各种屏幕尺寸
### 5️⃣ 强大的代理功能
通过工具启动的服务（如Flask），可以通过代理访问，避免跨域问题，轻松下载图片和文件。

## 📦 安装方式
### 方式一：直接运行（推荐）
下载 ScriptManager.exe 后，双击即可运行：

1. 程序会自动启动本地服务器
2. 浏览器会自动打开管理界面
3. 默认访问地址： http://localhost:3000
### 方式二：源码运行
```
# 克隆项目
git clone <项目地址>

# 安装依赖
npm install

# 启动服务
npm start
```
